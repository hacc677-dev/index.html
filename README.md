# PromptPilot demo

Mở `index.html` để vào trang đăng nhập demo.

## Gợi ý chạy ổn định
Nếu trình duyệt yêu cầu, dùng một local server đơn giản:

```bash
python -m http.server 8000
```

Sau đó mở:

```text
http://localhost:8000/index.html
```
